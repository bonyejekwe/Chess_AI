Classes:
-Board
	8x8 array
	Valid moves (out of bounds, piece already there etc.)
	MovePiece
	Check/checkmate
	Change pawn
	Castling

-Piece
	Get position, team methods
	Generalized move method

-King, Queen, Knight, Rook, Bishop, Pawn
	Inherits methods from Piece class 
	Valid move(input position) checks to see if given the piece it can move there

-Exceptions
	Invalid move/capturing

-Display board
	Displays board

AI:
-Game class
	-Position evaluator/Board Evaluator
		Evaluates the board or the certain positions of the pieces to try and make the best move.

	-Move generator
		Generates a legal move from the set of best moves.
